If you want to clone this repository, first create a database named inventory_ms in your localhost database (for example, MySQL in XAMPP). After cloning the project, go to both the client and server folders, copy the .env.example file, and rename the copied file to .env. Then open the .env file inside the server folder and change the database name to inventory_ms. After that, open 2 terminals in VS Code. In the first terminal, type cd client and run npm install. In the second terminal, type cd server and run composer install. When the installation is finished, stay in the server terminal and run php artisan key:generate. If it is successful, run php artisan migrate. After the migration is done, run php artisan serve in the server terminal, then go to the client terminal and run npm run dev. The project should now work properly. After opening the system, create a new account to start using the application. Thank you! 🙌
